(Rule-based Binding Affinity Prediction)

FEATURE: RULE-BASED BINDING AFFINITY INDEX

Implement a computational Binding Affinity Index (BAI) that estimates relative interaction strength using already-calculated structural interaction features.

⚠️ This is NOT experimental ΔG prediction and should be presented as a relative affinity score.

SCORING MODEL

Compute the Binding Affinity Index using the following weighted formula:

BAI =
(1.5 × hydrogen_bond_count)
+ (2.0 × salt_bridge_count)
+ (1.0 × hydrophobic_contact_count)
+ (0.2 × van_der_waals_contact_count)
− (0.5 × average_interaction_distance)


Normalize the final score to a 0–100 scale

Categorize binding strength as:

Weak (0–40)

Moderate (41–70)

Strong (71–100)

BACKEND IMPLEMENTATION

Use the existing interaction analysis results as input

Implement a utility function:

calculateBindingAffinityIndex(interactionStats)

Ensure calculations are fast and server-safe

Return results as JSON

  }
